# Future System Upgrade Paths: Yuba College
**Target:** IT / Computer Science Qualifications
**Status:** Backlog (Phase Two)

## Option 1: The Direct Hardware/Security Patch (Fastest)
**Certificate:** Network Security (Certificate of Achievement)
**Total Load:** 13 Units
* **IT 43AR / COMSC 43AR:** Unix System Management (3 units).
* **IT 44AR / COMSC 44AR:** A+ Test Certification preparation (4 units).
* **IT 44NR / COMSC 44NR:** Network Plus (N+) Test Certification preparation (3 units).
* **IT 44CR / COMSC 44CR:** Security+ preparation (3 units).
* **Admin Note:** This is a pure, highly focused cybersecurity injection. It perfectly aligns with the current Security+ self-study protocols. 

## Option 2: The Full IT Infrastructure Build
**Degree:** Associate in Science (AS) in Information Technologies
**Total Load:** 25 Units required for the degree major
* Includes the core networking and operating system courses from Option 1 (Unix, A+, Network+).
* **IT 45AR:** Supporting Network Clients (3 units).
* **Computer Language:** Programming course (3 units).
* **Electives:** 9 additional units selected from any other courses in the IT series.

## Option 3: The Software Engineering/Developer Route
**Degree:** Associate in Science (AS) or AS-T (Transfer) in Computer Science
**Total Load:** 60 to 61 Units
* **Core Focus:** Object-oriented programming utilizing C++ or Java.
* **Architecture:** Computer Architecture and Assembly Language.
* **Data Management:** Introduction to Data Structures.
* **Math/Science:** Requires heavy processing power, including Calculus I & II, Discrete Mathematics, and a science sequence like Physics, Chemistry, or Biology.
* **Admin Note:** The AS-T version of this degree guarantees admission to the California State University (CSU) system if a Bachelor's degree becomes a future objective.
